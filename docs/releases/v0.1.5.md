# v0.1.5 - Registry Namespace Alignment

This release aligns the official MCP Registry metadata with the GitHub organization namespace recognized by the Registry.

## Changes

- Updates `mcpName` and `server.json#name` to `io.github.AdsTurbo/adsturbo-creative-mcp`.
- Updates `server.json` package metadata to version `0.1.5`.
- Keeps the MCP server local-only with no AdsTurbo API key, video generation, telemetry, credit consumption, or ad account access.

## Verification

```bash
npm test
npm pack --dry-run --registry=https://registry.npmjs.org/
mcp-publisher validate
```

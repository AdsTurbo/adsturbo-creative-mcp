# Contributing

Thanks for helping improve AdsTurbo Creative MCP.

This project focuses on local, no-cost planning workflows for AI video ads. Contributions should preserve the cost boundary:

- no AdsTurbo API calls by default
- no video generation
- no hidden telemetry
- no ad account access
- no credit-consuming workflow without explicit user-provided credentials

## Good contribution areas

- MCP client setup examples for Claude Desktop, Cursor, Codex, and other agent clients
- Better video ad brief templates
- UGC script structures
- Storyboard output improvements
- Local CLI ergonomics
- Multilingual examples
- Safety and compliance notes for ad creative planning

## Development

```bash
npm install
npm run build
npm test
```

## Writing guidance

- Use references for structure, pacing, and inspiration, not for copying protected creative work.
- Keep ad claims grounded in product information that can be substantiated.
- Prefer clear workflow examples over broad marketing claims.
- Keep search terms natural and relevant: MCP server, AI agents, UGC scripts, video ad storyboard, AdsTurbo prompt, local CLI.
